Adidas Data Insights with Power BI

Project Overview

    This project involves the creation of an interactive dashboard using Microsoft Power BI to analyze Adidas company data obtained from Kaggle. The main objective is to visualize Adidas's sales data, track market performance, and analyze key financial metrics to identify trends, insights, and business opportunities. The dataset includes comprehensive sales data, regional performance, and product-wise sales. Using Power BI, the data was cleaned, transformed, and visualized to provide an interactive experience. Key features include region-wise sales analysis, product performance, and financial insights. This dashboard can be leveraged by business teams to drive strategic decisions and optimize market performance.

Project Details

    The project involves the following steps:

  • Load Data: Imported the Adidas Sales Dataset into Power BI Desktop.
  • Organize & Prepare Tables: Structured and organized the data into meaningful tables for analysis.
  • Connect Tables for Data Model: Established relationships between tables to build a cohesive data model.
  • Calculation Measures: Created calculated measures to derive key performance indicators (KPIs) and metrics.
  • Dashboard Design: Designed an interactive and user-friendly dashboard with various visualizations including pie charts, line charts, bar charts, and pivot tables.
